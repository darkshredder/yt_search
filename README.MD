# HOW TO RUN

    ```bash
    $ docker-compose up
    ```

# API ENDPOINTS

    ```bash
    $ curl -X GET http://localhost:1337/api/yt_search_results/
    $ curl -X GET http://localhost:1337/api/yt_search_results?search=tea
    ```

    `http://localhost:8000/api/yt_search_queries/` -- Create search queries to run
